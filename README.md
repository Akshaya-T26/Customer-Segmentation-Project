# Customer Segmentation Dashboard

A simple interactive Customer Segmentation Dashboard built using Python, Streamlit, Pandas, Scikit-learn, and Plotly.  
This project helps analyze customer behavior and group customers into different segments based on income and spending patterns.


## Features

- Load customer data from CSV  

### Customer Segmentation:
- Perform clustering using KMeans algorithm  
- Dynamic selection of number of clusters  
- Segment customers into:
  - High Value Customers  
  - Average Customers  
  - Low Value Customers  

- Cluster-wise analysis of customers  
- Interactive visualizations  

- Customer Segmentation Scatter Plot  
- Segment Distribution Pie Chart  


## Technologies Used

- Python  
- Streamlit  
- Pandas  
- Scikit-learn  
- Plotly Express  


## Dataset Structure

The dataset contains the following columns:

- CustomerID  
- Age  
- Annual_Income  
- Spending_Score  


## Project Workflow

- Load dataset using Pandas  
- Select features (Annual Income, Spending Score)  
- Apply KMeans clustering algorithm  
- Assign cluster labels to customers  
- Categorize customers into segments  
- Visualize results using interactive charts  
- Display cluster analysis  


## Visualizations

- Customer cluster scatter plot  
- Segment distribution pie chart  
- Cluster-wise summary table  


## Key Insights

- Customers are grouped based on income and spending behavior  
- High value customers are identified for premium targeting  
- Low value customers help understand budget segment  
- Useful for marketing strategy and customer targeting  
- Provides data-driven business insights
